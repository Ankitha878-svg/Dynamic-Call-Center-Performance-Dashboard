Dynamic Call Centre Performance Dashboard | Excel

This project is a dynamic and interactive Call Centre Dashboard built using Microsoft Excel, designed to monitor and analyze the performance of call centre representatives (R01–R05) and extract actionable operational insights.

Key Features & Insights:
-> Evaluates performance at the representative level.
-> Highlights that high call volume ≠ high revenue, showing efficiency differences among representatives.
-> Identifies top performers in both conversion rate and value generation.
-> Detects peak call periods, helping optimize staffing and operations.
-> Tracks customer service rating trends, providing opportunities for targeted improvements.
-> Analyzes calling patterns across male and female customers.

Purpose:
This dashboard goes beyond visualization — it is a decision-support tool designed to:

-> Identify top and underperforming representatives.
-> Optimize call centre operations and staffing.
-> Improve overall customer satisfaction through data-driven insights.

Technologies Used:
Microsoft Excel (PivotTables, Charts, Conditional Formatting, Formulas)
Dynamic and interactive dashboard elements for real-time decision-making
